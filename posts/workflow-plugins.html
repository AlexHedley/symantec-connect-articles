Workflow Plugin

Symantec  Workflow Manager has a number of Plugins available from install.

The idea here is to document each plugin, much like the Components.

  • License Status
  • Generate Business Model
  • Create GUID
  • Encryption Keys Generator
  • Export Translation Strings Plugin
  • Component Report
  • Model Report
